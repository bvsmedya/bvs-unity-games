# BVS.MEDYA - Unity Oyun Projeleri

## Projeler

### 1. SpaceShooter (Uzay Nişancı)
2D uzay nişancı oyunu. Ok tuşları hareket, Space/Click ateş.

Scripts:
- PlayerShip.cs    → Oyuncu gemisi, hareket + ateş + can
- Bullet.cs        → Mermi (oyuncu ve düşman)
- EnemyShip.cs     → Düşman zigzag hareketi + ateş
- EnemySpawner.cs  → Artan zorlukla düşman üretimi
- StarBackground.cs → Kayan yıldız arka planı

### 2. PlatformOyunu (2D Platform)
Klasik 2D platform oyunu. WASD/Ok tuşları + Space zıplama.

Scripts:
- PlayerController.cs  → Hareket, zıplama, can sistemi
- CameraFollow.cs      → Yumuşak kamera takibi
- EnemyPatrol.cs       → Devriye düşmanı
- Collectible.cs       → Coin/yıldız toplama
- GameManager.cs       → Skor, sahne yönetimi
- UIManager.cs         → HUD, Game Over, Pause
- MobileJoystick.cs    → Dokunmatik joystick
- MobileInputManager.cs→ Mobil + klavye input

## Unity Kurulum
1. Unity Hub > New Project > 2D
2. Scripts klasörünü Assets/ altına kopyala
3. Rehber dosyalarına bak
